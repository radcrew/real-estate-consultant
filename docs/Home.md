# Real Estate Consultant — Wiki

Internal MVP for an **AI-assisted commercial real estate search** workflow.

---

## What it does

| Capability | Description |
|---|---|
| **Intake** | Conversational wizard (guided form or AI chat) collects search criteria |
| **Search** | Criteria matched against ingested listings with ranked results |
| **Outreach** | Draft broker emails generated by LLM — no auto-send |
| **Watchlist** | Users can save and track listings |
| **Admin** | Submission review and ingestion management |

---

## Monorepo layout

```
real-estate-consultant/
├── frontend/          # Next.js app (UI, auth, intake, results)
├── backend/           # FastAPI API (search, LLM orchestration, auth)
├── services/
│   └── ingestion/     # Ingestion microservice (listing connectors)
├── .github/workflows/ # CI/CD pipelines
└── scripts/           # One-off dev utilities
```
