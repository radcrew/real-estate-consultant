# Explicitly out of scope (MVP)

Use this list to **prevent scope creep** during the first release. Items here may become future phases.

| Exclusion | Notes |
|-----------|--------|
| **Public listing marketplace** | No consumer-facing portal competing with major listing sites |
| **Full autonomous outreach** | No auto-send email or auto-submit contact forms |
| **CRM integrations** | No Salesforce, HubSpot, etc. in MVP |
| **Enterprise roles and permissions** | Beyond a single admin / internal user gate |
| **Billing and subscriptions** | No payment or plan management |
| **Native mobile app** | Web-first; responsive layout is in scope if the stack supports it |
| **Full GIS / market scoring engine** | Light geography and fit signals yes; full spatial analytics product no |
| **LoopNet-style product or prohibited sources** | No clone of major paywalled portals; no scraping of disallowed or private data |

If a feature is not listed here and not in [mvp-scope.md](mvp-scope.md), treat it as **undefined** and confirm with product owners before building.
